# Privacy Policy — Dua_Time / KTR

Last updated: May 2026

## About This Application
KTR is an internal automation tool developed by Dua_Time for content creation purposes. This application uses the Pinterest API to search and retrieve publicly available images as visual references for AI-generated media content.

## Data Collection
This application does not collect, store, or share any personal data from users or third parties. No user accounts, personal information, or private data are accessed or stored.

## Pinterest API Usage
We use Pinterest API with read-only access to search publicly available pins by keyword. This data is used solely as visual style references for internal content production. No Pinterest user data is collected or retained.

## Third-Party Services
This application integrates with:
- Pinterest API (read-only, public content only)
- Higgsfield AI (image and video generation)

No personal data is transmitted to or from these services.

## Data Security
As no personal data is collected, there is no risk of data breach or unauthorized access to user information.
